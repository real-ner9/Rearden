# Project Brief — Rearden

## Vision
Rearden is a **video-first hiring platform** that reimagines recruitment through short-form video profiles, Instagram-style candidate discovery, and real-time chat between recruiters and candidates.

## Core Requirements
1. **Video profiles** — candidates upload portrait video introductions (9:16 format)
2. **Reels-style feed** — vertical snap-scroll video feed for candidate discovery
3. **Instagram-style profiles** — tabs for Posts, Video, Vacancies on each candidate
4. **Real-time chat** — WebSocket-based messaging between recruiters and candidates
5. **Smart search** — text search + filters (skills, location, experience, availability)
6. **Resume pages** — formatted text resume view with download option
7. **Dark theme** — glass morphism design system, not LinkedIn-like

## Target Users
- **Recruiters** — browse candidates via feed/search, read profiles, chat, download resumes
- **Candidates** — register with video, post updates with hashtags, list vacancies they seek

## Non-Functional Requirements
- Mobile-first responsive design (bottom tab nav on mobile)
- Fast page transitions with Motion animations
- TypeScript across the entire stack
- Monorepo with shared types
