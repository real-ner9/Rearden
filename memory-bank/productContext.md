# Product Context — Rearden

## Why This Project Exists
Traditional hiring platforms (LinkedIn, Indeed) rely on text-heavy profiles. Rearden shifts the focus to video, letting candidates show personality and communication skills upfront, while giving recruiters a faster, more engaging way to evaluate talent.

## Problems It Solves
- **For recruiters**: sifting through hundreds of identical text resumes → watch 30-second video intros instead
- **For candidates**: standing out in a sea of profiles → video-first presence with personality
- **For both**: slow email chains → real-time chat with typing indicators and auto-replies

## How It Should Work

### Recruiter Flow
1. Land on homepage → see value proposition
2. Browse **Feed** (Reels-style vertical scroll) to discover candidates passively
3. Use **Search** with filters to find specific skills/locations
4. Open **Candidate Profile** → watch video, read posts, check vacancies
5. View **Resume** page for detailed background
6. **Chat** with candidate directly from profile or chat page

### Candidate Flow
1. **Register** with name, email, bio, skills, video upload, optional resume upload
2. Profile becomes discoverable in search and feed
3. Posts appear on their profile (hashtag-based content)
4. Receive chat messages from recruiters

## UX Goals
- Feel like Instagram/TikTok, not LinkedIn
- Dark theme with glass morphism — modern, premium feel
- Mobile-native experience with bottom tab navigation
- Smooth animations on every interaction (page transitions, tab switches, card hovers)
- Notification popup instead of separate page
